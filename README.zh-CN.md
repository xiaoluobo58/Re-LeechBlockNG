# Re: 从零开始的LeechBlock NG

[English](README.md) | **简体中文**

[![Awesome Humane Tech](https://raw.githubusercontent.com/humanetech-community/awesome-humane-tech/main/humane-tech-badge.svg?sanitize=true)](https://github.com/humanetech-community/awesome-humane-tech)

**Re: LeechBlock NG** 是 **LeechBlock NG (Next Generation) Chrome** 的一个分支，旨在帮助用户减少干扰、提升专注力，并持续引入新的功能与改进。

在继承原项目稳定可靠设计的基础上，Re: LeechBlock NG 将逐步加入：

* AI 功能
* i18n支持
* 更多新功能

你只需设置需要屏蔽的网站以及屏蔽时间，Re: LeechBlock NG 就能帮助你减少网络干扰，更专注于学习和工作。

## 开发

构建项目之前，请确保已将最新版 jQuery UI 安装到 `jquery-ui` 子目录中。

## 路线图

未来计划包括但不限于：

* AI 智能网站分类
* AI 专注建议
* 更多语言支持
* 更完善的统计与数据分析
* 更现代化的设置界面
* 持续兼容最新版 Chrome、Firefox 等浏览器

欢迎提交 Issue、Pull Request 或提出新的想法，一起让 Re: LeechBlock NG 变得更好。
